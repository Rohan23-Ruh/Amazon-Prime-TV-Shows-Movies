Amazon Prime TV Shows and Movies - Exploratory Data Analysis (EDA)

Project Overview

This project focuses on analyzing Amazon Prime Video's Movies and TV Shows dataset using Python. The objective is to explore content trends, understand audience-focused characteristics, and generate meaningful business insights through data cleaning, exploratory data analysis (EDA), and data visualization.
The analysis helps identify patterns related to content type, genres, release years, ratings, runtime, popularity, and audience preferences, enabling data-driven decision-making for content strategy.

Problem Statement

Analyze Amazon Prime Video's Movies and TV Shows dataset to identify content trends, patterns, and insights that support business decisions related to content planning, audience engagement, and platform growth.

Business Objective

Understand the distribution of Movies and TV Shows.
Identify popular genres and content categories.
Analyze release trends over the years.
Explore audience ratings and content performance.
Examine relationships between runtime, seasons, ratings, and popularity.
Generate actionable insights using data visualization.
